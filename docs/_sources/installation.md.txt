## Installation

### Stable version

Install PIASO to your environment:

```
pip install piaso-tools
```

### Development version

Install PIASO from GitHub directly:
 
```
pip install git+https://github.com/genecell/PIASO.git
```

### Environment set-up for PIASO tutorials

```
conda create --name piaso_env python=3.10 -y
conda activate piaso_env
conda install git pip -y
pip install piaso-tools
```

Quick validation:

```
python -c "import piaso; print('Success: PIASO installed and imported correctly!')"
```

Installing ipykernel for jupyter compatibility:

```
conda install anaconda::ipykernel
python -m ipykernel install --user --name piaso_env --display-name "Python piaso_env"
```

Additional packages required for tutorials:

```
pip install 'scanpy[leiden]'
pip install scrublet
```
